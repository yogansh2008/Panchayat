 🏡 Panchayat App

**Panchayat** is an AI-powered community management and digital governance app designed for housing societies and residential communities.

It brings together residents, admins, service providers, and security staff into one smart platform to manage daily society operations efficiently.

---

## 🚀 Features

### 👨‍💼 Admin Features
- Create and manage societies with unique code
- Add residents and service providers
- Generate and manage society guidelines
- Manage complaints and events
- Billing and accounting system
- Facility and room management

### 🧑‍🤝‍🧑 Resident Features
- Join society using code
- View services (plumber, electrician, etc.)
- Raise anonymous complaints
- RSVP for events
- Pay bills and track dues
- Book facilities and rooms
- Add visitor gate passes

### 🛠️ Service Provider Features
- Join society
- Add experience and service details
- Set availability status

### 🛡️ Security Features
- Track visitor entry and exit
- Manage gate passes

---

## 🤖 AI Assistant
- Answers user queries based on society data
- Uses guidelines, services, and events
- Provides instant and accurate responses
- No random or incorrect answers (data-based)

---

## 📋 Society Guidelines System
- Admin answers structured questions
- System auto-generates rulebook
- Clean and categorized display for users

---

## 📦 Tech Stack

- **Frontend:** React Native (Expo)
- **Backend:** Firebase
- **Database:** Firestore
- **AI:** Google Gemini / OpenAI API
- **Maps:** Google Maps API
- **Notifications:** Firebase Cloud Messaging

---

## 🧠 How It Works

1. Admin creates a society and generates a unique code  
2. Residents join using the code  
3. Data is linked using `societyId`  
4. Users interact with features (services, complaints, events, etc.)  
5. AI assistant provides instant help using stored data  

---

## 🔐 Key Concepts

- Role-based access (Admin, Resident, Service Provider, Security)
- Society-specific data isolation using `societyId`
- Anonymous complaint system
- Real-time data with Firebase
